GestureModel class!¶

class GestureModel.GestureModel(name_model, path, num_samples, th, create=True)¶

This class is used to save the parameters of a model

Parameters:
  • name_model – name of the model
  • path – In which folder the model will be saved, or is saved?
  • num_samples – number of samples by model
  • th – threashold of the model
  • create – If True then the model is created (GMR + GMM), if False the the model is only defined

The .txt files used to create the models must be named as: mod(i).txt

Where \(i=\{1,..,num\_samples\}\).
