class Classifier.Classifier¶

Class used to classify a gesture

analyzeActualWindow(window, numSamples, delay=0)¶

AnalyzeActualWindow separates the gravity and body acceleration features contained in the window of real-time acceleration data, by first reducing the noise on the raw data with a median filter and then discriminating between the features with a low-pass IIR filter.

Parameters:
  • window – windows to be processed and analalized
  • numSamples – is equal to the windows size
  • delay – delay of samples
